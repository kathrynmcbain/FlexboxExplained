Flex Direction

Flex-direction allows us to specify the direction we want blocks to display. We can stack them vertically by specifying 'flex-direction: column;'.
There are other ways we can use flex-direction.
'flex-direction: row;' will display the blocks in order horizontally.
'flex-direction: row-reverse;' will do the same but in reverse order - so 3,2,1.
'flex-direction: column-reverse;' will display as above, but in reverse order again.
